Refelction

One project I want to highlight from my resume is my Finance Tracker that I built with Flutter, because this project shows a lot of growth in my abilities. When starting this project in my second year of college, I knew the fundamentals of Python and Java very well. I knew how to work in both languages very well and could do what was asked of me for classes, but I hadn't worked with many frameworks before this. When I took this project as a challenge, I had to learn Flutter syntax, functions, how to render an app, how to update variables, how to test functions, and so much more to get this project to work. In a month's time I had watched hours of tutorials on Flutter and practiced within the confines of the project itself. However, I was able to build a prototype app that was functional and did what it was intended within strict parameters, but it worked. That was a massive achievement that I had been able to learn a new framework to be able to build something real, which made me hungrier to learn and advance my knowledge. An unexpected output from this was that I realized later, after abandoning the project due to school and other factors, was that it would be an introduction in a way to doing web development with HTML and CSS. I wanted to revisit the project later, but I wanted to find a more reproducible way that I could show off since I couldn't publish the app publicly. This led me to HTML, CSS, and JavaScript to make webpages that essentially do the same thing as Flutter, but could be publicly accessible with platforms such as GitHub Pages, which hosts this very webpage. My assumptions about building apps in Java were that it would be more accessible since all of my college career, I had heard that most apps and websites we use today are built in Java, and that Java is multi-platform, so I assumed it would be readily able to move from the fundamentals to app development. I was horribly wrong, and while Flutter is not a complicated framework, it is essentially a new language within Java, and you need to learn the way it works and relates to Java to be able to use it. In the end, this was a great experience to have and showed me just how much I could learn and push myself while still coming out on top. This gave me a taste for what the field would look like, having to learn a new framework fast and having to be able to apply it on the go.
